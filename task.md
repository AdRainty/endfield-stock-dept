# 终末地调度卷交易模拟系统 - 开发任务列表

## 需求分析

### 业务背景
基于《明日方舟：终末地》游戏设定，创建一个调度卷交易模拟系统。调度卷是游戏中的资源，玩家在两个交易所（四号谷底、武陵）进行交易。

### 核心特性
1. **双交易所**：四号谷底、武陵，各自独立行情
2. **档口交易**：类似券商的买卖五档行情
3. **实时价格**：价格持续波动，无集合竞价
4. **权限分离**：普通用户交易/管理员分配资源
5. **新手福利**：注册赠送每个交易所 10W 调度卷
